# Symbol-matching-Game

## Setup Instructions

1. Clone the repository:

    ```bash
    git clone https://github.com/hrideo/Symbol-matching-Game.git
    ```

2. Change to the repository folder

    ```bash
    cd Symbol-matching-Game
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

4. Run the project:

    ```bash
    node project.js
    ```
